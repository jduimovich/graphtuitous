This is a demo app

It has one endpoint which will return a JSON object 
 
      { 
        "version": 1.0, 
        "colour": "green",
        "compute": "function_run()"
      }

Version and colour are for demo apps to use to show change in outer and innerloop.

Compute is a performance helper for observability. A computation will be be performed on the server to simulate heavy loads or slow. The function will be shown here.
