This is a demo app

It has one endpoint at /test which will return a JSON object with the following fields. 
      { 
        "hostname":"johnpc"
        "version": 1.1, 
        "colour": "blue",
        "count": 1,
        "stack": "runtime used in demo", 
        "param": 1,
        "loop": 1,
        "compute": "fib(n)",
        "time": 1
      }

The fields returned are generally for demo purposes, described here. 

hostname
Track which pod executed the service on kubernetes.

version
Used to track which version was executed.

colour
Used to implement graphing for demos to show live edit, and outerloop changes.

count
Returns the number of times this server has returned a response. Can be reset to zero as services scale. .

stack
Name of the runtime used to implement the service. Quarkus, node.js, java

param
Parameter to function. Currently hardcoded, to be changed to pass as argument. 

loop
Loop count on how many times function is run. Currently hardcoded, to be changed to pass as argument. 

time
Time to run loop number of functions on the server. Can be changed to simulate heavy loads or slow.
